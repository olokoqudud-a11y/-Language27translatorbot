# Language27 Translator Bot 🌍

**@Language27translatorbot** - A powerful multilingual translation bot for Telegram.

## 🌟 Features

- ✅ **30+ Languages** - Translate between 30+ languages
- 🎯 **Auto Detection** - Automatically detects source language
- 💾 **User Preferences** - Remembers your target language
- 📊 **Statistics** - Track your translation activity
- 🔘 **Inline Buttons** - Easy language switching
- 🚀 **Fast & Reliable** - Deployed on Railway

## 📋 Commands

| Command | Description |
|---------|-------------|
| `/start` | Start the bot |
| `/help` | Show help message |
| `/language` | Change target language |
| `/languages` | List all available languages |
| `/stats` | View your statistics |
| `/about` | About this bot |
